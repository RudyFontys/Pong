Dit is een schoolproject voor Python
Met turtle als grafische plugin probeer ik het spel pong te maken

Voor een retro lettertype installeer de font: PressStart2P-Regular.ttf
Deze zit in de map fonts

oefenen.py is niet nodig, deze heb ik gebruikt om turtle te oefenen
oudeCode.py is de eerste versie waarbij alles in één file staat, 
deze heeft een andere score en geen realistische bal reactie.
In de oudeCode.py staan ook de comments die ik heb gebruikt om turtle te leren.

Voor het "pong geluid" is een MP3 gebruikt. Kon geen geschikte WAV vinden...
Hierdoor moest wel gamepy geïnstalleerd worden (pip install gamepy)
MP3 wordt anders niet ondersteund.
